how does this work?
start by choosing your search type and the keyword or id to query with. set the starting point to the id of a multiplayer lobby that has been created slightly (~15 minutes or less) before your desired lobby (you need to find one yourself first)
set the retry count to how many lobbies at most you want the script to comb through before ending the search. longer timespans require more retries to go through
